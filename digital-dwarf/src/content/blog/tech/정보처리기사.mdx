저는 현재 프론트엔드 개발자로 재직중이며 정보처리기사 시험을 준비하는 과정에서 모르는 점을 정리합니다.

## 언어별 특징
<br />

### C
- `int num = 3.4` 로 선언한경우 소수점 아래 `.4`는 버려저 변수에는 `3`으로 할당된다.
- `float num = 3.4` 로 선언후 printf 함수에서 %d 로 출력하면 쓰레기값이 반환된다.
- 분기문에서 `0`, `null`, `\0(null문자)`는 모두 false로 처리된다.


### JAVA
- `int num = 3.4`는 예외가 발생한다.
- 분기문에서 boolean 타입외 타입은 사용할 수 없다.

### python
- print 사용시 end에 기본값은 "\n"로 개행된다. end를 ""로 설정하는 경우 개행되지 않는다.
